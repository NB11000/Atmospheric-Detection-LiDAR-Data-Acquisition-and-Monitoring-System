## 一、背景（Background）

数据采集子系统已具备原始电压采集、跨进程 MMF 分发（CoreDataBus）、检测告警（Detection 线程）等基础设施，但 `StructuredSample` 中的 Vis（能见度）和 Cn²（折射率结构常数）字段始终为占位值 0.0。激光雷达大气检测的核心价值——从原始回波电压反演出大气光学参数——尚未实现。

### 问题（Problem）

#### 1. 缺少反演算法配置模型

反演算法需要 11 个可配置参数（增益均衡系数、K 常数、接收孔径、路径长度、滑动窗口帧数等），但系统缺少对应的配置 POCO 和 JSON 绑定逻辑，参数散落或硬编码。

#### 2. 缺少能见度（Vis）反演能力

需要从单通道或双通道回波电压中计算整帧大气能见度。核心算法为 Fernald (Klett) 后向积分法，涉及暗电流扣除、距离平方校正、远半段线性回归等步骤。

#### 3. 缺少 Cn² 剖面反演能力

Cn²（折射率结构常数）需从双通道信号的闪烁方差中统计得出，依赖滑动窗口（N=100 帧）累积足够样本后才能输出有效值。单通道模式下 Cn² 无意义。

#### 4. 预处理流水线缺失

原始电压必须经过暗电流扣除 → 近场盲区掩蔽 → 距离平方校正 → 双通道增益均衡四步预处理后，才能送入反演算法。缺少任一环节都会导致反演结果严重偏离。

#### 5. 反演结果未接入数据总线

AD_Controlcs 的 Analysis 线程在逐采样点填充 `StructuredSample` 时，Vis 和 Cn² 字段直接赋 0.0，反演器未集成到采集流水线中。

## 二、解决方案（Solution）

### 整体思路

**配置驱动 + 单帧能见度 + 滑动窗口 Cn²** —— 从 DeviceConfig.json 读取算法参数，在 Analysis 线程 for 循环之前同步调用反演器，Vis 为整帧单值、Cn² 为逐距离门剖面值，结果直接写入 `StructuredSample`。

### 具体实施

#### 1. 新建 `LidarAlgorithmConfig` 配置类（+41 行）

```csharp
public class LidarAlgorithmConfig
{
    public double GainEqualizationCoefficient { get; set; } = 1.0;
    public double KConstant { get; set; } = 4.48;
    public double ReceiverApertureD_m { get; set; } = 0.2;
    public double PathLengthL_m { get; set; } = 1000.0;
    public int Cn2WindowFrames { get; set; } = 100;
    // ... 共 11 个参数
}
```

`ConfigHelper.ReadLidarAlgorithmConfig()` 从 `DeviceConfig.json` 的 `LidarAlgorithm` 节绑定，注册为 DI 单例。

#### 2. 新建 `LidarInverter` 核心反演器（+307 行）

暴露单一入口：

```csharp
public (double vis, double[] cn2Profile) Invert(Voltage_block voltageBlock, byte chSel)
```

内部 5 步处理流水线：

```
原始电压 V1/V2
  → 步骤1: 暗电流估计与扣除（帧尾部均值，阈值钳位 < 1e-12）
  → 步骤2: 近场盲区掩蔽（r < BlindZone 置零）
  → 步骤3: 距离平方校正（V × r²）+ 双通道增益均衡（CH2 × coeff）
  → 步骤4: Fernald 能见度反演（远半段 ln(S) vs r 线性回归，α = -slope/2，Vis = 3.912/α）
  → 步骤5: Cn² 剖面反演（滑动窗口 N=100 帧，σI² = Var(ln(V1/V2))，Cn² = K × σI² × D^(7/3) × L^(-11/6)）
```

**哨兵值约定**：
- Vis < 0 → 反演失败（信噪比不足）
- Cn² = -1.0 → 数据无效（窗口未满 或 单通道模式），物理 Cn² 恒正，-1 天然区分

#### 3. 嵌入 Analysis 线程

在 `AD_Controlcs.Analysis()` 的 for 循环之前调用：

```csharp
double vis;
double[] cn2Profile;
try
{
    (vis, cn2Profile) = _lidarInverter.Invert(voltageBlock, chSel);
}
catch (Exception ex)
{
    _logger.LogError(ex, "LidarInverter.Invert 异常");
    vis = -1.0;
    cn2Profile = new double[count];
    Array.Fill(cn2Profile, -1.0);
}

// for 循环内：
detArr[i].Vis = vis;                              // 整帧共享
detArr[i].Cn2 = i < cn2Profile.Length ? cn2Profile[i] : -1.0;  // 逐距离门
```

带 try-catch 保护，反演异常时回退哨兵值，不影响采集主链路。
